# Notifications

Permission Slip can notify you through multiple channels when agents submit approval requests. Configure your preferences in **Settings → Notifications**.

## Available Channels

### Email

Receive email notifications when actions need approval. Requires a **contact email** set in your account settings.

### Web Push

Browser push notifications for real-time approval alerts. Works even when the Permission Slip tab is in the background. Your browser will ask for permission the first time you enable this.

### SMS

Text message notifications for approval requests. Requires a **phone number** in E.164 format set in your account settings.

## Enabling and Disabling

Each channel has a toggle button showing **Enabled** or **Disabled**. Click to toggle.

<Warning>
  If you enable Email or SMS notifications without the corresponding contact info (contact email or phone number), a warning will appear reminding you to add it in the Account section above. Notifications can't be delivered without valid contact information.
</Warning>

## Real-Time Dashboard Updates

Regardless of your notification settings, the dashboard always updates in real time via server-sent events. New approval requests appear instantly without refreshing the page.
