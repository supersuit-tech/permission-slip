# Activity Log

The activity log is a chronological record of everything that happens in Permission Slip — approval decisions, action executions, agent events, and more.

## Accessing the Log

Navigate to **Activity** from the sidebar, or click **View All Activity** at the bottom of the Recent Activity card on the dashboard.

## What's Logged

Every significant event is recorded:

- **Approval decisions** — requests approved or denied, with who decided and when.
- **Action executions** — actions that were executed (successfully or not).
- **Standing approval usage** — each time a standing approval is used.
- **Agent events** — registrations, deactivations.
- **Credential changes** — credentials stored or removed.

## Filtering

Use the filters at the top of the page to narrow results:

| Filter | Options |
|--------|---------|
| **Outcome** | All, Approved, Denied, Executed, Failed |
| **Event type** | Free-text search for specific event types |
| **Agent** | Filter by a specific agent |

Filters are reflected in the URL, so you can bookmark or share filtered views.

## Reading the Log

Each entry shows:

- **Timestamp** — when the event occurred (hover for relative time).
- **Agent** — which agent was involved.
- **Action** — the action type.
- **Outcome** — color-coded badge showing the result.

Outcome badges:
- **Approved** — green
- **Denied** — red
- **Executed** — blue
- **Failed** — red

## Pagination

The log loads a page of results at a time. Click **Load More** at the bottom to fetch older entries.

## Dashboard Summary

The **Recent Activity** card on the dashboard shows the most recent events with the same filtering options. It's a quick glance at what's happening without navigating away from the main view.
